# Description
Trovo (Trusted Resource for Omics and Variant Overview) is a hackathon winning project for analysing and retrieving information about proteins from various trusted sources like UniProt, AlphaFold DB, ChEMBL all displayed in a beautiful interface and integrated with an AI chatbot for follow up questions.

# Tech Stack
**Frontend** : HTML, CSS, JavaScript, 3Dmol.js  
**Backend** : Python Flask, Jinja  
**APIs** : UniProt, AlphaFold DB, ChEMBL, Google Gemini
